# focus

![Demo](./demo/demo.gif)

> [!IMPORTANT]
> Currently only working on MacOS because of the used notification system.

# Installation

```shell
  $ git clone https://github.com/tuxikus/focus.git
  $ cd focus
  $ cargo install --path .
```

# Usage

```shell
  $ focus 10m # set the focus timer to 10 minutes
  $ focus 3600s # 3600 seconds
  $ focus 2h # 2 hour timer
```
